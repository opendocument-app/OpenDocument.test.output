What else

This document shows you a little demo of what OpenDocument Reader is able to do. Additionally, you're going to find out what we're working on and when you will be able to edit your documents (hint: never ever).


You can use one of the following apps to open documents:

- Dropbox

- Gmail

- File Expert

- ASTRO File Manager

- Android Browser

- EStrongs File Explorer

- Box.net

- many more we don't know about...

Features

- zoom

- copy text

- formatting

- password-protected documents

- fast as hell

- image support

- spreadsheet support

- presentation support

- semi-official support for various other file types, like docx, pdf, etc.

- ads! More on that later...


Support for presentations is on the way but editting documents won't happen anytime soon. Sorry.


But... OpenDocument Reader is open source. Open sauce? No, open source! That means everybody's able modify it, improve it and help us make it even better. If you think you can do that drop us a mail.
